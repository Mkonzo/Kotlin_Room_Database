# 📝 Room Note Database (Kotlin)

## 📌 Overview
This project is a simple **Room Note Database simulation** built using **Kotlin**. It demonstrates the basic concepts of the **MVVM (Model-View-ViewModel) architecture** by separating data handling, business logic, and UI-related operations.

The project uses a custom implementation of a DAO layer to manage notes and demonstrates how a Repository communicates with a ViewModel to update data.

## 🚀 Features

- ✅ Add new notes
- ✅ Retrieve all notes
- ✅ Search notes by title
- ✅ MVVM architecture structure
- ✅ LiveData state management
- ✅ Repository pattern implementation
- ✅ Clean separation of responsibilities
